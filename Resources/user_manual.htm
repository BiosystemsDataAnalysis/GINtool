Configuring the tool

Select regulon file

The first step is to define which file contains the relation between regulon, gene names, BSU numbers etc. They can easily be downloaded from http://subtiwiki.uni-goettingen.de/ 

 

 Select the file by using the select operon file (see figure above). You have to define the once, every next time you start Excel this setting will be remembered.

Load the data

Because the regulon file has been defined, the data can be loaded into memory. This speeds up the processing during calculations. For this you need to select the "load reference data" button (see figure below).

 

Every time you start Excel and want to use the GINtool you have to press this button. Unless the data is loaded, most buttons are disabled by default.

Map the columns in the data file

 

When the data is loaded, all butons are enabled. First the column mapping between the regulons, the BSU codes and gene names have to be defined. By default, the subtwiki csv file has the following mapping:

Regulon file GINtool
locus_tag bsu
regulator regulon
gene gene
mode direction column

Direction

To indicate the direction of the mode of regulation (activate/repression) we already have defined the column that contains these mappings in the previous step. Because not everything is/was uniformly defined you first have to select what description is associated with up or down regulation. This is done by pressing the "regulon direction map". A dialog pops-up in which you can define what is up and down regulated (see Figure below).

 

In the table below the correct mapping between description and regulation is listed. Similar to the other settings these definition only have to be defined once. When the regulon file is changed however, these values need to be redefined.

Fold-Change (FC) ranges, p-value cut-off

Divide the FC range in high, middle and low, whereby FC under the low FC threshold are considered non-regulated. If you want to use one cutoff value, fill in the same value for all three ranges. To define relevance value select a cut-off for a p-value or related confidence value

Apply the tool

Now everything is defined, the tool can finally be deployed on the the data. Your data should be organized in at least 3 columns, the first containing p-values, the second Fold Changes, and the third the BSU numbers. Select the data in these 3 columns (excluding headings, see figure below) and click ‘apply to range’.

 

There are two modes of selecting which genes to include. Either by using the lower FC value of the p-value cut-off. To select which use the switch as shown in the figure below.

 

Results

The results are displayed over 3 sheets: Combined_1, Summary_1 and Mapped_1.

Mapped_1

This sheet indicates to which regulons the genes belong. The colour red indicates that the regulator is a repressor and green indicates an activator.

Summary_1

This sheet summarizes the activity of regulons by indicating the number of genes regulated divided in the different fold categories. The genes that are displayed have been selected based on the indicated p-value or fold-change cut-off values. The columns ‘Total number in database’ and ‘Total number in dataset’ indicate the total number of genes that are part of the regulon and that are found in the Regulon sheet from Subtiwiki and in the actual transcriptome dataset, respectively. The columns ‘Observed Counts and directions’ list the number of genes in the different fold-change categories. The column ‘Total Relevant’ is the sum of all genes that are above the ‘low’ fold-change threshold level, and therefore considered to show a relevant change in expression. The columns under ‘Percentage’ indicate the number of genes as percentage of the ‘Total number in dataset’ . The columns under ‘Logical direction’ indicate the change, expressed in percentage, that a regulon is activated or repressed. This is determined based on the ‘mode’ of regulation of genes and the number of genes that show a Fold Change larger than the Low FC range threshold.

The Combined_1

This sheet integrates the data from the previous two sheets. It indicates the regulators that regulate the gene, the most likely activity of the regulator expressed in %, whereby an upwards or downwards pointing arrow indicates activation or repression, respectively . The next percentage shows the total (‘-tot’) number of genes of the regulon that are regulated this way. For some regulators the ‘mode’ (activation/repression) is unknown and in these cases the first percentage is not shown. Finally, the Combined_1 sheet also indicates to which operons the gene belongs.

Operon (optional)

Optionally, the mapping between the genes and the operon can be included in the output. For this we need to select an extra file operon.csv (see figure below) that contains these mappings. This file can be downloaded from the subtwiki pages and contains only one single column so no extra mapping is necessary.
